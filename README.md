# Clima-Now
Site de pesquisas de clima por cidade desenvolvida com React, Typescript e Tailwind. Consumindo a API da HG Brasil.

# Weather App ☀️

## Funcionalidades

- Busca de cidades
- Mostra temperatura atual
- Umidade
- Previsão do tempo
- Ícones dinâmicos
- Loading state
- Tratamento de erros
- Responsivo

## Tecnologias

- React
- TypeScript
- Tailwind CSS
- Vite
- HG Brasil API
- Lucide React
